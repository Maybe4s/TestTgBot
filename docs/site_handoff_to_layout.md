# Prostohome: передача сайта в верстку/доработку

Дата: 2026-06-15

## Результат, который нужен от исполнителя

Первая рабочая версия сайта Prostohome, готовая к тестовому рекламному трафику и дальнейшему SEO.

## Основной документ

Исполнитель работает по `docs/site_tz_draft.md`.

## Состав работ первой очереди

1. Исправить каталог:
   - два дивана перенести из прямых диванов в малые формы;
   - в прямых диванах выставить порядок по типам;
   - убрать полосу в хедере каталога;
   - проверить карточки на мобильном.
2. Исправить визуал:
   - пуф поставить под углом 45 градусов;
   - футер уменьшить примерно в 2 раза;
   - убрать лишний воздух и визуальный шум.
3. Собрать/обновить главную:
   - hero;
   - `ХОЧУ / МОГУ`;
   - `8 причин купить стул`;
   - модельный ряд;
   - производство;
   - гарантия;
   - отзывы;
   - FAQ;
   - контакты.
4. Добавить B2B-блок по Бруку.
5. Проверить базовую аналитику и цели.

## Что нужно запросить у Андрея

1. Данные по Корвету:
   - размеры;
   - материалы;
   - варианты ткани/цветов;
   - механизм;
   - наполнение;
   - цена;
   - фото;
   - преимущества;
   - условия поставки.
2. Финальные вводные по Бруку для B2B:
   - цена или условия;
   - наличие;
   - цвета;
   - объемы;
   - условия для дилеров/дизайнеров.
3. Подтверждение порядка категорий и карточек.

## Что считается готовым

- Все правки первой очереди внесены.
- Страницы проверены на desktop и mobile.
- CTA ведут в согласованные каналы.
- Метрика установлена.
