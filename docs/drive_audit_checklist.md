# Диски и материалы: чеклист проверки оформления

Дата: 2026-06-15

## Цель

Понять, что уже есть на дисках, что оформлено нормально, чего не хватает и что можно добавить для работы сайта, рекламы, канала и подрядчиков.

## Что проверить

1. Структура папок:
   - сайт;
   - каталог;
   - фото товаров;
   - видео;
   - логотипы;
   - отзывы;
   - документы;
   - реклама;
   - подрядчики;
   - архив.
2. Названия файлов:
   - модель;
   - цвет;
   - ракурс;
   - дата;
   - версия.
3. Доступы:
   - кто может смотреть;
   - кто может редактировать;
   - есть ли публичные ссылки;
   - нет ли лишнего доступа.
4. Материалы для сайта:
   - логотип;
   - фото главной;
   - фото карточек;
   - характеристики;
   - отзывы;
   - юридические данные.
5. Материалы для рекламы:
   - баннеры;
   - короткие видео;
   - офферы;
   - UTM/ссылки;
   - посадочные.
6. Материалы для канала:
   - обложки;
   - рубрики;
   - исходники;
   - готовые посты.

## Что добавить

- папку `00_На согласование`;
- папку `01_Готово к публикации`;
- папку `02_Исходники`;
- папку `03_Архив`;
- файл `README` с правилами названий;
- таблицу учета материалов: файл, назначение, статус, ссылка, ответственный.

## Критерий готовности

Есть понятная структура, подрядчик может найти нужные материалы без переписки, а для сайта и рекламы видно, какие файлы готовы и какие еще нужно доснять/дособрать.
