# Статус активной цели

Дата: 2026-06-15

## Главное

Ключевой результат цели — согласованное ТЗ по сайту и передача в верстку. По этому направлению подготовлен предфинальный пакет:

- `docs/site_tz_draft.md` — ТЗ по сайту Prostohome.
- `docs/site_handoff_to_layout.md` — короткая передача исполнителю.
- `owner_uploads/prostohome_site_marketing_brief_2026-06-14.md` — рабочий бриф по сайту, SEO и Директу.
- `owner_uploads/prostohome_site_work_kp_2026-06-14.md` — КП/этапы работы с сайтом.

## Готово к согласованию

1. Структура первой версии сайта.
2. Срочные правки каталога и визуала.
3. B2B-блок по Бруку.
4. Список материалов для Андрея.
5. Критерии приемки первой версии.
6. Базовые SEO-требования.
7. Чеклист аналитики и заявок.

## Что блокирует фактическую передачу в верстку

1. Нет подтверждения цены Гренады.
2. Нужно сверить привязку 16 карточек к моделям.
3. Не подтверждены цвета по моделям.
4. Не хватает точных данных и фото Axiom/Vesper.
5. Не хватает данных по Корвету.
6. Нужно утвердить финальную формулировку B2B-баннера Брук.
7. Нужно подтвердить каналы заявок.
8. Нужен доступ к Яндекс.Киту или Chrome CDP для живых правок.

Подготовлен файл `docs/drive_audit_checklist.md`. Для фактической проверки нужен доступ к дискам/папкам и список ссылок.
