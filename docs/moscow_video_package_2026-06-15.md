# Видео по объекту на улице Московской

Дата: 2026-06-15

## Текущий статус

В рабочей папке проекта видеофайлы по объекту не найдены. Есть фото и голосовые, но нет `.mp4`, `.mov`, `.m4v`, `.avi` или `.mkv`. Поэтому фактический монтаж ролика пока невозможен без загрузки исходников.

## Ответ на 15 июня

Текст для отправки:

> По Московской сейчас собираю ролик из остаточного материала. Для нормальной версии нужны исходники видео: общие планы объекта, процесс, детали и финальный результат. Если части кадров не хватает, соберу черновик из доступных остатков и отдельно отмечу, какие кадры нужно доснять или заменить.

Короткая версия:

> По Московской беру в работу остатки и собираю черновик. Если по исходникам будет провал в логике ролика, сразу отмечу, каких кадров не хватает.

## Минимальный ролик из остатков

Формат: 9:16, 30-45 секунд.

Структура:

1. 0-3 сек: самый сильный кадр результата или общий вид объекта.
2. 3-8 сек: адрес/объект без лишней детализации, титр "Объект на Московской".
3. 8-20 сек: процесс работ, 3-5 коротких кадров.
4. 20-32 сек: детали качества, узлы, аккуратность, материалы.
5. 32-42 сек: финальный вид.
6. 42-45 сек: контакт или призыв написать для расчета.

## Титры

Вариант 1:
- Объект на Московской
- Работа по остеклению
- Аккуратный монтаж
- Чистый результат
- Напишите, рассчитаем ваш объект

Вариант 2:
- Московская
- Замер, подготовка, монтаж
- Детали решают результат
- Готовый объект
- Денис Суворов, остекление

## Что нужно найти или загрузить

Обязательные кадры:
- общий вид объекта до/после;
- 2-3 кадра процесса;
- крупные планы деталей;
- финальный результат;
- вертикальная версия или видео с запасом по краям для 9:16.

Желательно:
- 5-10 секунд живого движения на объекте;
- кадр с мастером/Денисом;
- короткий отзыв или голосовой комментарий;
- фото/видео проблемного места до работ.

## Если исходников мало

Собрать не финальный рекламный ролик, а короткий отчет:

1. Титр "Объект на Московской".
2. Слайд/кадр с тем, что сделано.
3. Нарезка доступных деталей.
4. Финальный кадр.
5. Текст "Полный кейс подготовим после добора кадров".
