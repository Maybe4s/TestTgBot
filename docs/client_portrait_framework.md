# Портрет клиентов после запуска

Дата: 2026-06-15

## Когда делать

После запуска сайта/рекламы и после отпуска, когда появятся реальные обращения, заявки и диалоги.

## Источники данных

- заявки с сайта;
- звонки;
- сообщения в Telegram/WhatsApp;
- обращения из Авито;
- комментарии и реакции в канале;
- данные Метрики;
- отчеты колл-центра;
- база директолога.

## Сегменты для проверки

1. Розничный покупатель для дома.
2. Семья с детьми.
3. Владельцы животных.
4. Дизайнеры интерьера.
5. Комплектаторы квартир/объектов.
6. Шоурумы и мебельные партнеры.
7. Строительные/ремонтные компании.

## Что собрать в первую неделю после запуска

- 30-50 реальных обращений, если объем позволит.
- Статусы обращений: целевой/нецелевой/теплый/горячий.
- Частые вопросы.
- Частые возражения.
- Источники заявок.
- Какие товары спрашивают чаще.
- Какие формулировки приводят к ответу.

## Выходной результат

Документ с 3-5 рабочими портретами клиентов, которые можно использовать для сайта, рекламы, Авито, канала и скриптов колл-центра.
